List of Commands Available

For a more comprehensive guide, visit our main website.

Command Description Format
Add Creates a new task and add it to our list. add NAME [by DEADLINE] [from START_TIME to END_TIME] [repeat PATTERN FREQUENCY] [tag TAG...]
List Lists out only certain tasks that fulfill criterias/filters. list [TYPE...] [by DEADLINE] [from START_TIME] [to END_TIME] [tag TAG...] [sort SORTING_ORDER]
Show Further filters the listed tasks by criterias/filters. show [TYPE...] [on DATE] [by DEADLINE] [from START_TIME] [to END_TIME] [tag TAG...]
Hide Excludes certain tasks that met the criterias/filters. hide [TYPE...] [on DATE] [by DEADLINE] [from START_TIME] [to END_TIME] [tag TAG...]
Find Searches tasks with given certain name (this will work even if you do not know the entire full name). find KEYWORD...
Update Changes details pertaining to a task. update INDEX [name NAME] [by DEADLINE] [from START_TIME to END_TIME] [repeat PATTERN FREQUENCY] [tag TAG...]
Complete Marks a task as done. complete INDEX
Delete Removes a task from the list. delete INDEX
Undo Reverses the action(s) made by you. undo [last STEPS]
Redo To "reverse" the undo. redo [last STEPS]
Clear Removes all tasks from the list. clear
Switch List Opens another task list (if it does not exist, a new task list will be created). switchlist FILENAME
Rename List Renames the current task list. renamelist FILENAME
Relocate Moves all task lists to a different folder/directory. relocate PATH
Collapse Shrinks all tasks into a single row, only showing the name of the tasks. collapse
Expand Reveals the details for all the tasks. expand
Calendar Changes the view of the calendar on the right side of the UI. calendar day|week|today
Help Displays this screen. help
